Propriedades de Position no CSS

Esta página demonstra as diferenças entre as propriedades de posição do CSS: relative, absolute, fixed e sticky.

Exemplo de Posição: Relative

A posição relative move o elemento em relação à sua posição original.

Este elemento está usando position: relative.

Exemplo de Posição: Absolute

A posição absolute posiciona o elemento em relação ao elemento pai mais próximo com posição definida.

Este elemento está usando position: absolute.

Exemplo de Posição: Fixed

A posição fixed fixa o elemento em relação à janela do navegador, independentemente de rolagem.

Este elemento está usando position: fixed.

Exemplo de Posição: Sticky

A posição sticky permite que o elemento se comporte como relative até atingir um ponto específico na rolagem, onde ele se torna fixo.

Este elemento está usando position: sticky.
