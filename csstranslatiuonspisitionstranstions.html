Entendendo CSS Transitions

As CSS Transitions permitem que você adicione animações suaves quando as propriedades CSS de um elemento mudam. Isso melhora a experiência do usuário, tornando mudanças visuais mais agradáveis.

Exemplo 1: Transição de Escala e Cor

Código CSS:

.box {
    width: 100px;
    height: 100px;
    background-color: #4CAF50;
    transition: all 0.5s ease;
}
.box:hover {
    transform: scale(1.2);
    background-color: #8BC34A;
}

Exemplo 2: Transição de Opacidade

Código CSS:

.box-opacity {
    opacity: 1;
    transition: opacity 0.5s ease;
}
.box-opacity:hover {
    opacity: 0.5;
}

Exemplo 3: Transição de Borda

Código CSS:

.box-border {
    border: 2px solid transparent;
    transition: border-color 0.5s ease, border-width 0.5s ease;
}
.box-border:hover {
    border-color: #8BC34A;
    border-width: 5px;
}

Exemplo 4: Transição de Rotação

Código CSS:

.box-rotate {
    transform: rotate(0deg);
    transition: transform 0.5s ease;
}
.box-rotate:hover {
    transform: rotate(45deg);
}

Resumo

Com esses exemplos, você viu como aplicar CSS Transitions a diferentes propriedades como escala, opacidade, borda e rotação. Teste os exemplos acima e explore como transition pode tornar seus sites mais dinâmicos e interativos!
