Entendendo CSS Flexbox

O Flexbox é um modelo de layout no CSS que permite criar layouts dinâmicos e flexíveis com facilidade. Ele organiza elementos em contêineres flexíveis, ajustando o espaço e o alinhamento de acordo com o design desejado.

Como ativar o Flexbox?

Para usar Flexbox, basta aplicar display: flex; ao contêiner pai. Veja um exemplo básico:

.container {
    display: flex;
}

Propriedades principais do Flexbox

O Flexbox tem propriedades para o contêiner pai e os itens filhos. Aqui estão as principais:

1. Propriedades do contêiner:

  • flex-direction: Define a direção dos itens (linha ou coluna). Exemplos: row, column.
  • justify-content: Alinha itens no eixo principal. Exemplos: flex-start, center, space-between.
  • align-items: Alinha itens no eixo transversal. Exemplos: flex-start, center, stretch.

2. Propriedades dos itens filhos:

  • flex-grow: Define o quanto um item pode crescer em relação aos outros.
  • flex-shrink: Define o quanto um item pode encolher se o espaço for limitado.
  • flex-basis: Define o tamanho inicial de um item antes de distribuir o espaço.

Veja como essas propriedades funcionam em um exemplo:

    display: flex;
    justify-content: center; /* ou space-between, space-around, etc. */
}

1
2
3
.container {
    display: flex;
    align-items: center; /* ou flex-start, stretch, etc. */
}

Exemplo 4: Crescimento dos itens (flex-grow)

1
2
3
.item {
    flex-grow: 1; /* Ajusta o quanto o item cresce */
}

Resumo

O Flexbox é uma ferramenta poderosa e versátil para criar layouts responsivos e bem organizados. Experimente os exemplos acima para entender como cada propriedade afeta o design!
